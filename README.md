# Grahak Khata - Goldsmith Ledger Web App

Grahak Khata is a digital ledger web application specially designed for goldsmith and jewellery businesses to manage girvi items, customer records, and daily account entries in a simple and organized way. It makes data entry faster and helps maintain business records more efficiently than manual registers.

## Features

- Add and manage customer details
- Record girvi items with related information
- Track transactions and account history
- View and update records easily
- Clean and simple interface for daily business use
- Responsive design for desktop and mobile devices

## Tech Stack

- HTML
- CSS
- JavaScript
- React
- Vite
- Vercel

## Live Demo

[Open App](https://grahak-khaata.vercel.app)

## Project Overview

This project was built for a real jewellery business client to simplify girvi item entry, customer management, and hisaab-kitab. The goal was to replace manual record-keeping with a faster and more convenient digital system that can be used in day-to-day operations.

## Installation

```bash
git clone https://github.com/kgsharma7417/grahak-khaata.git
cd grahak-khaata
npm install

```
